## 为什么写博客

作为一个信息安全专业的学生，学习过程中有太多东西值得记录。无论是 CTF 比赛中遇到的巧妙思路，还是调试了一整天才修好的 BUG，又或者是读完某本技术书后的感悟——这些东西如果不写下来，很快就会遗忘。

博客是一个很好的载体：

1. **知识沉淀**：用自己的话把学到的东西讲清楚，是最好的学习方式
2. **技术积累**：一年后回头看，你会惊讶自己成长了多少
3. **建立连接**：写博客是认识同好的最佳方式

## 这个网站的技术选型

这个博客使用了以下技术栈：

- **[Astro](https://astro.build/)**：静态站点生成器，性能极佳，Markdown 原生支持
- **Vercel**：免费部署平台，自动 CI/CD
- **纯 CSS**：没有使用任何 CSS 框架，从头手写，精简高效

选择 Astro 而不是更流行的 Next.js，是因为：

1. 博客本质上是内容站点，不需要客户端 JavaScript 框架
2. Astro 默认零 JS 输出，出海更快
3. 学习曲线平缓，写 Markdown 就行
